# **App Name**: Aedura Elite

## Core Features:

- Centralized School Setup: A comprehensive setup module that allows admin users to input school details, define academic years, manage classes/sections, assign subjects, and configure timetables within a unified interface.
- AI-Powered Attendance Insights: Generates a report for teachers on overall student attendance trends, potential truancy, and suggest interventions using AI tool.
- Multi-User Role Management: User Management allows to create users with roles for Teachers, Students, Parents and Admins and assign specific permissions to them, guaranteeing the security and hierarchical accessibility of functionalities. Supports Single Sign On with Google and Microsoft Entra.
- Comprehensive Fees Management: Handles every fee process from construction, invoices, record keeping, report generations, and receipts within an intuitive design to ease operation of payments.
- Streamlined Communication Hub: Centralizes announcements, messaging, and SMS/email communications into a unified hub, allowing efficient information dissemination to various user groups.
- Dynamic Student Dashboards: Tailored dashboards that provides quick student profile review, upcoming assignments submission deadlines, exam results overview, along with general fee status overview and notice publishing sections.
- Smart Library: Manage the existing book repository, automate issue tracking and generate smart insights to track borrowing behavior.

## Style Guidelines:

- Primary Colors:
  - Primary 500: `#4C6EF5` (Soft Royal Blue)
  - Primary 400: `#748FFC`
  - Primary 600: `#3B5BDB`
- Neutral / Background:
  - Background: `#F5F7FB` (Soft White-Blue)
  - Surface Glass: `rgba(255, 255, 255, 0.18)`
  - Card Stroke (glass border): `rgba(255, 255, 255, 0.35)`
  - Text Primary: `#1A1A1A`
  - Text Secondary: `rgba(26, 26, 26, 0.65)`
- Accents:
  - Accent Gold: `#E9C46A`
  - Success: `#2F9E44`
  - Warning: `#F4A261`
  - Error: `#E63946`
- Shadows (for soft 3D buttons):
  - Light Shadow: `rgba(255, 255, 255, 0.8)`
  - Dark Shadow: `rgba(0, 0, 0, 0.12)`
- Font: 'Inter' for all headings and body text, maintaining a consistent minimalist luxury aesthetic. Note: currently only Google Fonts are supported.
- Body text: Light font weight, lower opacity (70-80%) for readability and to reinforce minimalist aesthetic.
- Use Phosphor Icons with a light weight throughout the platform.
- Apply glassmorphism to cards with subtle background blur, low opacity, and a subtle border to achieve a soft depth effect.
- On page load: fade from 0 to 100% opacity with a slight upward movement for all components.
